# 🎯 The First Attacker — Tactical Goalkeeper Analysis

A professional data-driven dashboard focusing on the tactical impact of modern goalkeepers. This project analyzes how elite keepers contribute to team build-up through progressive passing, sweeping actions, and shot creation.

## 🚀 Key Features
- **Tactical Rankings**: Identify the best playmaking goalkeepers.
- **Scouting Profiles**: Deep-dive into individual tactical patterns.
- **Strategic Matrix**: Visualize the trade-off between aggressive sweeping and progressive distribution.

## 📊 Metrics Explained
- **Progressive Passes**: Passes that move the team significantly closer to the opponent's goal.
- **Sweeping Actions**: Defensive actions taken outside the penalty area.
- **SCA (Shot Creating Actions)**: The two offensive actions leading directly to a shot.

## 🛠️ Tech Stack
- **Dashboard**: Streamlit
- **Visualization**: Plotly
- **Data Processing**: Pandas, NumPy

## 🏃 How to Run Locally
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
